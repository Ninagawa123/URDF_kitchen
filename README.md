# URDF_kitchen beta
<img width="600" alt="urdf_kitchen_beta" src="docs/urdf_kitchen_beta.png">

URDF_kitchenは、URDFの組み立てをサポートするツールです。
STLファイルにジョイントポイントを設定し、ノードで接続していくことで組み立て、URDFとしてエクスポートします。
重量やイナーシャ計算、パーツごとの着色などにも対応しています。
